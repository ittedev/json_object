# json_object

JSON object types and helper functions

# Example

```ts
import { JSONObject } from 'https://deno.land/x/json_object/mod.ts'

const json = '{"text":"I understand","len":12}'

const data = JSON.parse(json) as JSONObject

console.log(data)
```
